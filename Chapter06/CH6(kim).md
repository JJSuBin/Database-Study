# 데이터 모델

⇒ 데이터 구조,제약조건,연산

# 관계 데이터 연산

⇒ 관계 대수 , 관계 해석

# 관계 대수

⇒원하는 결과를 얻기 위해 데이터의 처리과정을 순서대로 기술함

⇒ 집합관련 연산자를 사용 & 릴레이셩에 연산을 적용해도

릴레이션(폐쇄적인 특성)

# 일반집합 연산자

⇒ 합집합

⇒ 교집합

⇒ 차집합

⇒ 카티션 프로덕트(두 릴레이션의 모든투플을 연결하여 생성된 투플)

# 순수 관계 연산자

⇒ 릴레이션의 구조와 특성을 이용함

⇒ 셀렉트(릴레이션에서 조건을 만족하는 투플)

⇒ 프로젝트(only 릴레이션에서 주어진 속성을의 값)

⇒ 조인(공통 속성을 이용해 두 릴레이션의 투플을 연셜&생성)

⇒디비전 (나누어지는 릴레이션에서 나누는 릴레이션과 관련있는 투플)

# 확장된 관계 대수 연산자

⇒ 세미 조인 ⇒ 조인 속성으로 프로젝트 연산을 한 릴레이션을 이용해 조인 연산

⇒ 외부 조인 ⇒  자연 조인 연산에서 제외된 모든 투플을 결과 릴레이션에 포함 

# 관계 해석

처리를 원하는 데이터가 무엇인지만 기술함(결과값 도출)
